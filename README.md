# e-plantShopping

**Project Name:** e-plantShopping — Paradise Nursery Shopping Application

## Overview

Paradise Nursery is a React + Redux shopping cart application for an online houseplant store. Browse and shop for a wide variety of beautiful houseplants organized into categories, add them to your cart, and manage your order — all in one seamless experience.

## Features

- 🏡 **Landing Page** — Background image, company description (About Us), company name, and "Get Started" button
- 🌱 **Product Listing Page** — 30 unique houseplants across 5 categories, each with image, name, description, and price
- 🛒 **Shopping Cart** — Increment/decrement quantities, delete items, view subtotals and grand total, checkout
- 🔄 **Redux State Management** — Global cart state using Redux Toolkit

## Plant Categories

1. Air Purifying Plants (6 plants)
2. Aromatic Fragrant Plants (6 plants)
3. Insect Repellent Plants (6 plants)
4. Medicinal Plants (6 plants)
5. Low Maintenance Plants (6 plants)

## Tech Stack

- React 18 + Vite
- Redux Toolkit + React-Redux
- CSS (no external UI libraries)

## Getting Started

```bash
git clone https://github.com/YOUR_USERNAME/e-plantShopping.git
cd e-plantShopping
npm install
npm run dev
```

## Build

```bash

npm run preview
```

## Deploy to GitHub Pages
